# image-hotspot
Image hotspot feature for showing details about specific feature of product

## Installation
`npm i image-hotspot`

## Directions:

| Direction Name  | Value       |
| ------------- |:-------------:|
| Up left       |  up-left      |
| Up right      |  up-right     |
| Down left     |  down-left    |
| Down right    |  down-right   |
| Left Up       |  left-up      |
| Left down     |  left-down    |
| Right Up      |  right-up     |
| Right down    |  right-down   |
